1.Protocolos de comunicação em rede

Os protocolos de comunicação em rede são conjuntos de regras que estabelecem como os dados são formatados e processados, permitindo que os computadores se comuniquem entre si. Eles são responsáveis por coordenar a troca de informações entre dispositivos de redes diferentes, garantindo que todos falem a mesma linguagem.

2.Http

HTTP é a sigla para Hypertext Transfer Protocol, ou Protocolo de Transferência de Hipertexto, e é um protocolo de comunicação que permite a transferência de dados na internet

3. Ftp

FTP é a sigla para File Transfer Protocol, ou Protocolo de Transferência de Arquivos. Trata-se de um conjunto de regras que permite a transferência de arquivos entre computadores em uma rede TCP/IP, como a internet

4.SMTP

SMTP é a sigla para Simple Mail Transfer Protocol, que é um protocolo de comunicação padrão para o envio de e-mails pela internet.

5.Modelo TCP/IP

TCP/IP é um conjunto de protocolos que possibilita a comunicação entre computadores e servidores, formando a Internet. Ele combina o Protocolo de Controle de Transmissão (TCP), responsável pela transmissão de dados, com o Protocolo de Internet (IP), que identificar os computadores e servidores.
